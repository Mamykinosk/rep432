# Travel Memorize Android

## Идея проекта

**Travel Memorize** — приложение, которое представляет собой интерактивную карту для сохранения истории путешествий и обмена туристическим опытом.

Основная концепция заключается в визуализации посещённых мест через метки на карте, к которым пользователь прикрепляет информацию: фотографии, текстовые описания и даты посещения.

Приложение позволяет взаимодействовать с другими пользователями, подписываясь друг на друга. Это даёт возможность просматривать карты друзей, изучать их рекомендации и находить локации для новых маршрутов. Также пользователь может посмотреть список мест, которые он посещал.

## Базовый функционал

### Авторизация и профиль

- Регистрация и вход через email/пароль с обязательной валидацией полей
- Личный профиль со статистикой:
  - количество стран
  - количество городов
  - количество воспоминаний
- Настройки профиля

### Интерактивная карта

- Отображение текущего местоположения пользователя
- Добавление меток в местах посещения через кнопку **«Добавить»**
- Визуальное различие между своими метками и метками друзей

### Управление воспоминаниями

- Создание записи:
  - прикрепление нескольких фото
  - заголовок
  - дата
  - описание
- Просмотр деталей:
  - экран с галереей фото
  - текст воспоминания
  - открытие по нажатию на метку
- Редактирование записей
- Удаление записей

### Взаимодействие между пользователями

- Поиск друзей по имени / username
- Возможность подписаться на другого пользователя
- Возможность отписаться от пользователя через кнопку **Delete**
- Список друзей
- Переход на интерактивные карты друзей
